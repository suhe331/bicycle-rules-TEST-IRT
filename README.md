## 📌 概要

2026年4月から施行された自転車の青切符制度に対応した、  
小学生から大人まで使える自転車ルール学習ウェブアプリです。

**IRT（項目反応理論）× AI問題生成 × 間隔反復学習** を組み合わせ、  
「測るだけで終わらない」学習サイクルを実現します。

---

## ✨ 主な機能

| 機能 | 内容 |
|---|---|
| **アダプティブテスト** | 2パラメータIRTで個人の能力θを推定。Fisher情報量最大の問題を逐次選択 |
| **AI問題自動生成** | 路面写真・動画からVision AI＋LLMが問題・解説・IRT初期値を一括生成 |
| **間隔反復学習** | SM-2アルゴリズムで最適タイミングに復習を自動スケジュール |
| **ローカライズ問題** | 先生が地元の交差点を撮影→学校タグ付きで問題化→その学校の生徒に優先出題 |
| **弱点ダッシュボード** | カテゴリ別理解度・今日やること・復習スケジュールを可視化 |
| **オープンバッジ** | 習得済みカテゴリにIMS Global標準のデジタル認定証を発行（将来） |

## 🧮 使用する数理モデル

### ① 2パラメータロジスティックモデル（2PLM）
```
P_i(θ) = 1 / (1 + exp(−a_i · (θ − b_i)))
```
- `a`：識別力　`b`：困難度　`θ`：受験者の能力値

### ② EAP推定（回答のたびにθをリアルタイム更新）
```
θ̂ = Σ[θ·L(θ|u)·π(θ)] / Σ[L(θ|u)·π(θ)]
```
θを61点グリッドで離散化してブラウザJS上で計算（サーバー不要）

### ③ Fisher情報量（次問選択・終了判定）
```
I_i(θ) = a_i² · P_i(θ) · [1 − P_i(θ)]
SE(θ̂) = 1 / √(ΣI_i(θ̂)) < 0.35 → 測定完了
```

### ④ SM-2アルゴリズム（間隔反復）
```
EF_new = EF_old + 0.1 − (5−q)·(0.08 + (5−q)·0.02)
I(n) = I(n−1) × EF
```
初回正解→1日後、2回目→3日後、3回目→7日後…

## 🏗 システム構成

```
Frontend (Vercel)
├── クイズUI（IRT計算をブラウザJSで実行）
├── 学習ダッシュボード
└── 教員管理画面（写真アップ・問題承認・クラス進捗）

Backend (Supabase)
├── PostgreSQL（questions / users / responses / schools / badges / law_master）
├── Auth（先生：Google OAuth　生徒：学校コード＋ニックネーム）
├── Storage（路面写真・動画）
└── Edge Functions（問題選択ロジック・AI API呼び出し）

External API
├── Claude API（Vision AI＋LLM：問題生成時のみ）
├── Google OAuth（先生ログイン）
└── e-Gov 法令API（法改正の自動検知・将来）
```

**月額コスト（初期フェーズ）：実質 ¥0〜¥1,000**  
IRT計算はブラウザJSで完結→サーバー不要。AI APIは問題生成時のみ課金。
